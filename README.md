# 💗 Сайт "Моя Мама" 💗

Красивый и современный веб-сайт, посвящённый вашей маме!

## 📁 Содержание файлов

- **index.html** - Главная страница со всем контентом
- **styles.css** - Современное оформление с красивыми цветами и анимациями
- **script.js** - Интерактивность и анимации

## 🎨 Особенности дизайна

✨ **Современный UI Design**
- Красивые градиенты (розовый и красный)
- Плавные анимации и переходы
- Адаптивный дизайн (работает на телефонах, планшетах, ПК)
- Красивые иконки и эмодзи
- Эффекты при наведении мыши

🌟 **Секции сайта:**
1. **Навигация** - Быстрый переход по сайту
2. **Героя секция** - Впечатляющее приветствие с анимацией
3. **О ней** - 3 карточки с описанием качеств
4. **Её лучшие качества** - 6 пронумерованных качеств
5. **Галерея** - Место для ваших фото (4 места)
6. **Статистика** - Красивые цифры и факты
7. **Контакт** - Кнопка для отправки любви
8. **Футер** - Красивый нижний колонтитул

## 🚀 Как использовать

1. **Откройте файл index.html в браузере**
   - Двойной клик на файл или
   - Перетащите в браузер

2. **Настройте под себя:**
   - Измените текст в файле index.html
   - Добавьте свои фото вместо плейсхолдеров в галерею
   - Измените цвета в styles.css (переменные в :root)

## 🎯 Как добавить фотографии

В секции галереи найдите коды вроде:
```html
<div class="gallery-placeholder">
    <span>📸</span>
    <p>Ваше фото 1</p>
</div>
```

Замените на:
```html
<img src="path/to/your/photo.jpg" alt="Фото мамы">
```

## 🎨 Цвета (как их изменить)

В файле styles.css найдите `:root` и измените цвета:
```css
--primary-color: #ff6b9d;      /* Розовый */
--secondary-color: #c44569;    /* Красный */
--accent-color: #ffa502;       /* Оранжевый */
```
